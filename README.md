# gdou-jw-tools
## 海大教务系统助手

### 主要技术栈：
+ Python第三方验证码识别库
+ 前端：Jquery+HTML+CSS
+ 后台：SpringBoot+JPA
+ Java爬虫：Jsoup+HttpClient

### 拥有的功能：
+ 成绩查询
+ 考试查询
+ 一键评教
