# gdou-tools
## GDOU校园助手

### 主要技术栈：
+ 前端：JQuery、HTML、CSS
+ 后台：SpringBoot、JPA、MySQL
+ Java爬虫：Jsoup、HttpClient
+ Python爬虫：requests、BeautifulSoup
+ 其它主要Python库：pytesseract、Pillow、school_api、pyDes等

### 目前拥有的功能：
+ 成绩查询
+ 考试查询
+ 一键评教
+ 自动抢课
+ 今日校园自动签到
